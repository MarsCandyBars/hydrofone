# Hydrofone

## Description
A simple background noise app

## Noise Options
- White
- Brown
- Pink
- Oscillator

## Installation
```sh
1. Clone the repo
2. cd into the directory
3. Run npm install
4. Run ng serve -o
```

## Docker

Build and run:

```sh
docker build https://github.com/MarsCandyBars/hydrofone.git -t hydrofone
docker run --rm -d -p 8080:80 hydrofone
```
